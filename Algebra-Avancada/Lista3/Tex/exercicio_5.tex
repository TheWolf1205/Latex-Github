  Seja $\varphi : G \to G$ um homomorfismo de grupos. Mostre que:
\begin{enumerate}
    \item Se $g \in G$ e $o(g) < \infty$ então $o(\varphi(g))$ divide $o(g)$.
    \item Se $o(g) < \infty$ e $\varphi$ é injetivo então $o(\varphi(g)) = o(g)$.
    \item Se $H \leq G$ e $H$ é cíclico então $\varphi(H)$ é cíclico.
    \item Se $H \leq G$ e $H$ é abeliano então $\varphi(H)$ é abeliano.
    \item Se $H \trianglelefteq G$ então $\varphi(H) \trianglelefteq \operatorname{Im}(\varphi)$.
\end{enumerate}
